# lucasew's offtopic

[![GitHub Pages](https://github.com/lucasew/lucasew.github.io/actions/workflows/autorelease.yaml/badge.svg)](https://github.com/lucasew/lucasew.github.io/actions/workflows/autorelease.yaml)

Blog migrado de Hugo para Astro.

## Comandos

- Setup: `mise install && npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`
- Format: `npm run format`

## Estrutura de conteúdo

- Conteúdo em `src/content/` (mantido da estrutura original do Hugo)
- Rotas em formato `/:lang/...` (`/en/...` e `/pt/...`)
- Suporte a frontmatter existente, aliases, `index.{en,pt}.md`, `_index.{en,pt}.md`
